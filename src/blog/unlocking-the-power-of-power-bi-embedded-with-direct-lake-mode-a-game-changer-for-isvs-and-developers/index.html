What is Direct Lake Mode?

Direct Lake Mode is a new storage mode in Power BI that directly accesses Delta tables in OneLake, removing the need for data import or duplication.

This brings several advantages:

‍

Faster Performance

Directly loading parquet-formatted files into memory results in faster queries and reporting.

Real-Time Insights

Users get up-to-the-minute data without requiring scheduled refreshes.

Optimized Resource Usage

Avoids redundant data storage and import overhead.

Resiliency

Automatically falls back to DirectQuery mode when needed, ensuring continuity.

‍

For ISVs and developers using Reporting Hub, this means a faster, more responsive experience for their customers while reducing the burden of managing data refreshes.
