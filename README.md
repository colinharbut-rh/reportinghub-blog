# Reporting Hub Blog

Static blog hosted on Cloudflare Pages. Migrated from Webflow.

## Structure

```
/templates/                   → Header and footer partials (single source of truth)
  header.html
  footer.html
/src/blog/                    → Source HTML (with <!-- INCLUDE header --> / footer placeholders)
  index.html                  → Blog index
  [slug]/index.html           → Individual post pages
/blog/                        → Built output (generated by npm run build)
/scripts/build.js             → Inlines templates into src/blog and writes to blog/
/wrangler.toml                → Cloudflare Pages config
/worker/                      → Cloudflare Worker (routes /blog/* from main domain)
```

## Templates

Header and footer are shared across all blog pages. Edit once in `templates/header.html` and `templates/footer.html`, then run `npm run build` to regenerate `blog/`.

## Build

From the repo root:

```bash
npm run build
```

This reads every HTML file in `src/blog/`, replaces `<!-- INCLUDE header -->` and `<!-- INCLUDE footer -->` with the template contents, and writes the result to `blog/`. Run after changing templates or adding/editing posts in `src/blog/`.

## Deploy

Push to `main` → GitHub Actions → Cloudflare Pages (automatic). Ensure `npm run build` runs before deploy (e.g. in CI or Cloudflare Pages build command) so `blog/` is up to date.

## Images

All images hosted on Cloudflare R2 at `images.thereportinghub.com`.

## Routing

A Cloudflare Worker intercepts `/blog/*` on `thereportinghub.com` and proxies
to this Pages deployment. All other paths pass through to Webflow unchanged.

## Adding a post

1. Create `src/blog/[slug]/index.html` with the same structure as existing posts (include `<!-- INCLUDE header -->` and `<!-- INCLUDE footer -->` where nav and footer go).
2. Run `npm run build` to regenerate `blog/`.
3. Commit and push to `main`; live in ~30 seconds.
